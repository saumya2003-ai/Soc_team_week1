# 🔐 SOC Fundamentals and Operations Project

 Overview
This project demonstrates the implementation of a Security Operations Center (SOC) including monitoring, log analysis, threat detection, and incident response.

---

 Objectives
- Understand SOC architecture and workflow
- Perform log analysis and threat detection
- Implement alert rules
- Map attacks using MITRE ATT&CK framework

---

 Tools Used
- Wireshark (Network Analysis)
- Snort (IDS)
- Wazuh (SIEM/XDR)
- Kibana (Dashboard)
---

|-------------|------------|
| T1059 | Command Execution |
| T1078 | Valid Accounts |
| T1046 | Network Scanning |
